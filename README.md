# 📅 Bookr – Booking Management System

Bookr is a full-stack booking management application developed as part of an internship project at IAKK NKft. It allows users to manage reservations efficiently through a modern web interface backed by a secure REST API.

## 🚀 Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Angular + TypeScript |
| Backend | Java REST API |
| Database | MySQL (Stored Procedures) |
| Auth | JWT (JSON Web Tokens) |

## ✨ Features

- User authentication and authorization via JWT
- Create, view, update, and cancel bookings
- Role-based access control
- RESTful API communication between frontend and backend
- Stored procedure-based database logic for performance and maintainability

## 🗂️ Project Structure

```
bookr/
├── frontend/        # Angular application
├── backend/         # Java REST API
└── database/        # SQL scripts, stored procedures and scheduled events
```
